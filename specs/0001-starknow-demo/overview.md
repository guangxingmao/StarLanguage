# OpenSpec: 星知（StarKnow）Demo - Overview

Status: Draft  
Owner: TBD  
Last updated: 2026-01-23  

## Summary
面向儿童的通识教育 App Demo，核心是“趣味内容 + 轻量问答 + 简单对战 + 成长激励”，并支持跳转到抖音/哔哩哔哩观看视频内容。

## Goals
- 2-3 周内完成可演示 Demo
- 体验闭环：看内容 → 学知识 → 做题 → 得成长/成就
- 童趣、轻快、精致的 UI 体验
- 视频外链跳转可用

## Non-Goals
- 完整知识图谱与复杂推理
- 大规模题库与自动化推荐
- 完整社群生态
- 多端复杂账号体系

## Target Users
- 6-12 岁儿童
- 亲子共学场景的家长
